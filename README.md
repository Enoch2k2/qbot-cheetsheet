# QBot Cheetsheet
```
qbot in sync - logs tc in as sync support

qbot in dispatch - logs tc in as dispatch support

qbot out - logs out tc

qbot queue <link> notes - queues up student for pairing with tc

qbot take <link> - pairs student with tc

qbot unresponsive - if student is unresponsive for 5 minutess, this command will put them back in queue and the question will be ignored for 5 minutes

qbot done - resolves pairing and puts tc back in queue for another pairing

qbot resolve <link> - resolves question

qbot whoison? - lists all the tcs, also shows how long they have been waiting in queue or shows how long they have been in a pairing.

qbot q - list all the students in queue
```
